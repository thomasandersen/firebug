Issue 6841: Use platform API for CSS autocompletion

Testing

Steps to reproduce

  1. Open Firebug and select the HTML panel
  2. Inspect the blue <div> above
  3. Select the Style side panel
  4. Add the following properties and confirm the auto-completions:
    • text-transform: up ⇒ uppercase
    • -moz-appearance: no ⇒ none
    • transition: 1s height, 1s col ⇒ 1s height, 1s color
    • color: w ⇒ white
    • color: inac ⇒ inactiveborder
    • color: rg ⇒ rgb()
    • height: 50px ⇒ no completion
    • background-image: line ⇒ linear-gradient()
    • background: linea ⇒ linear-gradient()
    • background: bl ⇒ black
    • background: yellow bl ⇒ no completion
    • background: yellow rep ⇒ yellow repeat
    • image-rendering: opt ⇒ optimizeSpeed
    • pointer-events: no ⇒ none
    • pointer-events: vis ⇒ no completion
    • content: at ⇒ attr()
    • content: ur ⇒ url()
    • content: un ⇒ unset
    • font: cal ⇒ no completion
