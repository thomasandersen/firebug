  1. Open Firebug and select the HTML panel.
  2. Use the inspector to inspect the element below with the text "Inspect This Element".
  3. Select also the Style side panel. There should be one 'element.style' CSS rule.
  4. Click at the only property margin-right.
  5. Press up-arrow once. The field should now contain "margin-left".
  6. Press down-arrow once. The field should now contain "margin-right" again.
