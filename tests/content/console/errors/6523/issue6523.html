Issue 6523: JSD2: Show Stack Trace With Errors

Steps to reproduce

  1. Open Firebug
  2. Enable the Console and Script panel
  3. Reload the page
  4. Check Show Stack Trace with Errors option
  5. Click the test button
  6. There should be one error log in the Console panel: Error: b
    Expand it and check the stack trace. It should be:
    b() d() onExecuteTest() onclick()
