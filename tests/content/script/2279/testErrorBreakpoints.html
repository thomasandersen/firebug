Test for Error Breakpoints

  1. Enable Firebug and reload this page.
  2. Use this button to create an error
  3. You should see one error displayed in Firefox status bar.
  4. Open Firebug, select the Console panel.
  5. Create error breakpoint by clicking on the gray circle close to the error message.
    Just before: foo.bar += 'baz';
  6. Reload this page again.
  7. The debugger should stop at the line causing the error.
